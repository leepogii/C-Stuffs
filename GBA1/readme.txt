TOM & JERRY GAME
====================
=	HOW TO PLAY	   =
====================
1. Press START button to start the game.
2. 3 cats are trying to catch Jerry, and Jerry needs to find cheese without 
getting caught.
3. Press B to use special ability. Jerry will squeak and that will make cats run away (can be used only once).
4. Press SELECT button to go back to main menu anytime.
